# Myocardial_infarction_70k
This project predicts myocardial infarction risk using machine learning on a dataset of 70,000 records. Supervised models (Logistic Regression, Random Forest, Gradient Boosting, SVM, KNN) and unsupervised methods (K-Means, DBSCAN, Agglomerative Clustering) were applied to analyze patterns and evaluate predictive performance.
