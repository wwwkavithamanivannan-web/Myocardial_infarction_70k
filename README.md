# Predicting Myocardial Infarction Risk Using Machine Learning

## Project Overview
This project focuses on predicting the risk of myocardial infarction (heart attack) using machine learning techniques. Both supervised and unsupervised algorithms were applied to analyze patterns and predict cardiovascular risk based on clinical and demographic data.

The dataset used for this study contains **70,000 patient records**, providing a large sample for model training and evaluation.

## Machine Learning Approaches

### Supervised Learning Models
The following classification algorithms were implemented and compared for predictive performance:

- Logistic Regression  
- Random Forest  
- Gradient Boosting  
- Support Vector Machine (SVM)  
- K-Nearest Neighbors (KNN)

These models were trained to predict the likelihood of myocardial infarction based on input health features.

### Unsupervised Learning Models
To explore hidden patterns and group similar patient profiles, the following clustering algorithms were applied:

- K-Means Clustering  
- DBSCAN  
- Agglomerative Hierarchical Clustering

These methods help reveal natural clusters within the dataset that may correspond to different cardiovascular risk profiles.

## Model Evaluation
The supervised models were evaluated using standard classification metrics including:

- Accuracy  
- Precision  
- Recall  
- F1-score  
- Classification Report

## Tools and Technologies
- Python  
- Scikit-learn  
- Pandas  
- NumPy  
- Google Colab  

## Project Goal
This project demonstrates how machine learning can support early risk prediction and uncover hidden patterns in cardiovascular health data, contributing to data-driven healthcare analysis.
